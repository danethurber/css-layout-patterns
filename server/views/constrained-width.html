{{#markdown}} # Constraining an Element's Width Define an element max-width to prevent it from expanding to width ```scss .constrained-width { max-width: 400px; } ``` ## Center a Block Element Set Horizontal Margins to Auto to Center a Block Element ```scss .constrained-width { max-width: 400px; } .constrained-width--centered { margin-left: auto; margin-right: auto; } ``` {{/markdown}}
