{{#markdown}} # Floats Take the element out of the normal document flow and place it on the right or left of it's container. Text and inline elements will wrap around it. {{/markdown}} {{#markdown}} ## Clearing the float Floated elements stack against each other {{/markdown}} {{#markdown}} The clear property can be used to place the element after a previously floated element. {{/markdown}} {{#markdown}} ```scss .first_image { float: left; } .second_image { float: left; clear: left; } ``` {{/markdown}}
