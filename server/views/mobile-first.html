{{#markdown}} # Mobile First Target smallest screen first and build upon styles as screen gets bigger. Optimizes for mobile device hardware. {{/markdown}} {{#markdown}} ```scss .stoplight { background-color: green; height: 200px; } @media (min-width: 480px) { .stoplight { background-color: red; } } @media (min-width: 768px) { .stoplight { background-color: yellow; } } ``` {{/markdown}}
